# Changelog

All notable changes to Maoxuan Product Agent are documented here.

## [1.0.0] - 2026-07-10

### Added

- Chinese-first product decision skill for Codex, Claude Code, Cursor, and Agent Skills compatible tools.
- Reasoning engine distilled from complete readings of *On Practice*, *On Contradiction*, and related Volume I essays.
- 36 product, growth, operations, data, project, and organization playbooks.
- Chinese response examples, evidence grading, stage judgment, stop lists, and decision-switch conditions.
- 36-case self-test, four clean-session forward tests, and an automated quality gate.
- Universal and platform-specific install paths.
- Chinese GitHub Pages site, English index, machine-readable GEO files, and citation metadata.
